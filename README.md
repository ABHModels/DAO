# DAO: An Open-Source X-ray Reflection Model

**DAO** is a comprehensive, open-source model for simulating X-ray reflection spectra from photoionized accretion disks. It solves the radiative transfer equation using the robust Feautrier method and incorporates atomic data from the powerful XSTAR database.
